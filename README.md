# FeelingDiary

애플리케이션 이름 : 감성일기

애플리케이션 개요 : 물리 엔진을 사용하여 구슬을 이용한 감성적인 느낌을 최대한 살린 신개념 일기장, 자신의 폰을 움직이면 자이로센서를 통해 구슬이 움직이는 효과를 느낄 수 있으며 이로인해 사용자의 감성을 최대한 극대화 할 수 있는 것이 가장 큰 특징입니다.
